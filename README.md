# 🤖 TelegramConsoleBot

Minecraft Paper плагин для интеграции Telegram бота с сервером на версии 1.16.5

## 📋 Функции

✅ **Регистрация и вход** - /reg и /login команды
✅ **Привязка Telegram** - Связь Minecraft аккаунта с Telegram
✅ **Консольные команды** - Выполнение команд сервера через Telegram (админ)
✅ **2FA аутентификация** - Двухфакторная проверка через Telegram
✅ **Отслеживание IP** - История последних входов
✅ **Блокировка аккаунтов** - Временный бан через Telegram

## 🚀 Установка

1. **Скачай JAR файл** из релизов
2. **Поместить в папку `plugins`** на сервере
3. **Перезагрузить сервер** - `reload` или рестарт
4. **Отредактировать `config.yml`** в папке `ATGCON`
   - Вставить токен бота Telegram
   - Установить ID администратора

## 📝 Команды Minecraft

### Для всех игроков:
```
/reg <пароль>              - Регистрация аккаунта
/login <пароль>           - Вход в аккаунт
/link                      - Привязка Telegram аккаунта
/link <код>               - Завершить привязку
```

### Для администраторов:
```
/tgconsole reload          - Перезагрузка плагина
/tgconsole token <токен>  - Изменить токен бота
/tgconsole admin <ID>     - Установить админа
/tgconsole status         - Статус плагина
```

## 📱 Команды Telegram

### Для всех пользователей:
```
/start                     - Справка по командам
/link                      - Получить код привязки
```

### Для администратора:
```
/cmd <команда>            - Выполнить команду на сервере
```

## ⚙️ Конфигурация

Файл `ATGCON/config.yml`:

```yaml
telegram:
  bot-token: "ТВОЙ_ТОКЕН_ЗДЕСЬ"
  admin-id: 7742036100

security:
  min-password-length: 4
  enable-2fa: false
```

## 🔧 Процесс регистрации и привязки

1. Игрок вводит `/reg пароль` - регистрация аккаунта
2. Игрок вводит `/login пароль` - вход в аккаунт
3. Игрок вводит `/link` - получает информацию о привязке
4. Игрок напишет боту `/link` в Telegram
5. Боот выдаст код привязки (20 символов)
6. Игрок вводит в Minecraft `/link КОД`
7. ✅ Аккаунт привязан!

## 📊 Хранение данных

Плагин хранит данные в папке `ATGCON`:
- `players.json` - Данные игроков (пароли, привилегии, IP)
- `accounts.json` - Связанные Telegram аккаунты

## 🔐 Безопасность

- Пароли хешируются перед сохранением
- IP адреса отслеживаются и хранятся
- 2FA можно включить для дополнительной защиты
- Коды привязки имеют срок действия 1 час

## 📚 Зависимости

- **Java 8+**
- **Paper 1.16.5**
- **Maven** для сборки
- **TelegramBots** 5.7.1
- **GSON** для работы с JSON

## 🏗️ Сборка проекта

```bash
mvn clean package
```

JAR файл будет создан в папке `target/`
